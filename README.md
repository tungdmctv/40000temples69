# 40000temples69

Landing page สำหรับงาน

`พิธีถวายมหาสังฆทาน 40,000 กว่าวัดทั่วประเทศ ปีพุทธศักราช 2569`

จัดทำด้วย Nuxt เพื่อสรุปข้อมูลงานให้อ่านง่ายบนหน้าเดียว โดยรวมสาระสำคัญจากต้นฉบับ เช่น

- ภาพรวมงานและวันจัดพิธี
- กำหนดการ
- การร่วมบุญ
- การเดินทางและลงทะเบียนรถ
- พื้นที่นั่ง แผนผัง และการพักค้าง
- เอกสารดาวน์โหลด

## Tech Stack

- Nuxt 3
- Static generation for GitHub Pages
- GitHub Actions for automatic deployment

## Local Development

```bash
npm install
npm run dev
```

เปิดที่ `http://localhost:3000`

## Production Build

```bash
npm run build
```

หรือสำหรับ static output:

```bash
npm run generate
```

## Deployment

โปรเจกต์นี้ deploy อัตโนมัติด้วย GitHub Actions เมื่อมีการ push ไปที่ branch `main`
